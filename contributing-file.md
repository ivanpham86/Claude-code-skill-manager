# Contributing / Hướng dẫn đóng góp

Thank you for your interest! / Cảm ơn bạn đã quan tâm!

## Getting Started / Bắt đầu

```bash
git clone https://github.com/ivanpham86/Claude-code-skill-manager.git
cd Claude-code-skill-manager
npm install
npm run dev
```

## Pull Request Process / Quy trình PR

1. Fork and create a feature branch from `main`
2. Make your changes with clear commit messages
3. Test locally with `npm run dev`
4. Submit PR with description of changes

## Code Style / Quy ước code

- JavaScript/TypeScript with React
- Inline styles (no CSS framework)
- Electron IPC for frontend-backend communication
- Handler pattern for backend logic (`electron/handlers/`)

## Reporting Issues / Báo lỗi

Open an issue with:
- Steps to reproduce / Các bước tái tạo lỗi
- Expected vs actual behavior / Hành vi mong đợi vs thực tế
- OS and app version / Hệ điều hành và phiên bản app
- Screenshots if applicable / Ảnh chụp màn hình nếu có
